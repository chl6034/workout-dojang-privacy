🏋️ 운동도장

개인정보처리방침 (Privacy Policy)

1. 개요

운동도장(이하 "앱")은 사용자의 개인정보 보호를 중요하게 생각합니다. 본 방침은 앱이 수집하는 정보와 그 사용 방법을 설명합니다.

✅ 본 앱은 기본적으로 모든 데이터를 사용자의 기기 내부에만 저장하며, 사용자가 클라우드 백업 기능을 사용하지 않는 한 외부 서버로 전송하지 않습니다.

2. 수집하는 정보

운동도장은 다음 정보를 기기 내부에 저장합니다:

  • 운동 기록 (날짜, 운동 종류, 기록 시각)
  • 앱 설정 정보 (시간 표기 형식, 스탬프 설정, 알림 설정 등)
  • 사용자가 추가한 운동 종목
  • 사용자가 직접 업로드한 커스텀 스탬프 이미지

위 정보는 기본적으로 사용자의 기기에만 보관되며, 개발자 또는 제3자에게 자동으로 전송되지 않습니다.

3. 클라우드 백업 및 복원

앱은 Google 계정 연동을 통한 클라우드 백업/복원 기능을 제공합니다. 이 기능을 사용할 경우:

  • 운동 기록, 사용자 추가 운동 종목, 앱 설정이 Google 서버(Firebase)에 저장됩니다.
  • 커스텀 스탬프 이미지는 클라우드에 저장되지 않습니다.
  • 저장된 데이터는 본인의 Google 계정으로만 접근 가능합니다.
  • 클라우드 백업은 사용자가 직접 실행하는 경우에만 수행되며, 자동으로 전송되지 않습니다.

클라우드 백업 기능은 선택 사항이며, 사용하지 않아도 앱의 모든 기능을 정상적으로 이용할 수 있습니다.

4. 광고

본 앱은 Google AdMob을 통해 배너 광고 및 전면 광고를 제공합니다. AdMob은 광고 제공 및 최적화를 위해 다음 정보를 수집할 수 있습니다:

  • 광고 식별자 (Android: GAID / iOS: IDFA)
  • 기기 정보 (모델, OS 버전 등)
  • 광고 상호작용 데이터 (노출, 클릭)

iOS 14.5 이상에서는 앱 추적 투명성(ATT) 동의 팝업을 통해 광고 추적 여부를 선택할 수 있습니다.

광고 관련 자세한 내용은 Google 개인정보처리방침을 참고하세요.

5. 권한 사용

앱은 다음 권한을 요청할 수 있습니다:

  • 사진/갤러리 접근: 커스텀 스탬프 이미지 선택 시에만 사용
  • 알림: 운동 알림 기능 사용 시 (선택 사항)

권한은 해당 기능 사용 시에만 요청되며, 거부해도 핵심 기능은 정상 작동합니다.

6. 데이터 보관 및 삭제

기기 내 데이터는 앱을 삭제하면 함께 삭제됩니다. 클라우드에 백업된 데이터는 앱 내 계정 삭제 기능을 통해 삭제할 수 있습니다. 개발자는 사용자 데이터에 접근하거나 열람할 수 없습니다.

7. 제3자 제공

앱은 광고 서비스(Google AdMob)를 제외하고 사용자 데이터를 제3자에게 판매하거나 제공하지 않습니다.

8. 아동 개인정보 보호

본 앱은 만 14세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.

9. 방침 변경

개인정보처리방침이 변경될 경우 본 페이지를 통해 공지합니다. 변경 내용은 페이지 하단의 업데이트 날짜를 확인하세요.
